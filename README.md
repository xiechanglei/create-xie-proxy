# 一个简单的http服务器

use yarn 
```shell
# 在当前文件夹创建一个http服务器，端口是9123(默认)
yarn create xie-proxy 
```

use npm 
```shell
# 在当前文件夹创建一个http服务器，端口是9123(默认)
npm create xie-proxy
```

额外参数
```shell
# 在当前文件夹创建一个http服务器，并且指定端口
yarn create xie-proxy 9999
```
